# -
智能旅游推荐系统是一个面向游客与管理端的双端平台，覆盖景点浏览、行程规划、游记社区与订单管理，结合地图与智能推荐能力提升出行决策效率。项目亮点包括：自动获取景点的天气情况、引入地图api，获取景点的地图情况、支持对景点、酒店、特产、美食进行预订点赞评论、支持ai辅助自动进行行程规划、聊天室功能、混合推荐策略。


技术栈
- 前端（用户端）：Vue 3、Vite、Element Plus、Pinia、Vue Router、Axios、Socket.IO Client、V‑md‑editor、高德地图 JS API
- 前端（管理端）：Vue 3、Vite、Element Plus、Pinia、Vue Router、Axios、ECharts
- 后端：Node.js + Express、JWT 鉴权、Multer 文件上传、Socket.IO
- 数据库：MySQL（业务数据）+ MongoDB（游记社区）


用户端功能
- 账号体系 ：注册 / 登录 / 退出 / 个人资料编辑 / 头像上传
- 景点模块 ：景点列表、筛选与搜索、详情展示、评分与评论、收藏
- 行程规划 ：创建行程、选择景点、AI 行程建议、行程保存、分享链接、行程导出
- 游记社区 ：游记发布与编辑、图片上传、标签管理、点赞、评论、浏览统计
- 订单系统 ：门票 / 美食 / 酒店 / 特产下单、订单列表、订单详情、支付与取消
- 推荐与服务 ：个性化推荐、相似推荐、地图展示、天气信息
- 即时沟通 ：聊天室在线用户、消息发送、输入状态提示
-

管理端功能
- 管理员登录 ：后台登录、权限校验
- 内容管理 ：景点 / 美食 / 酒店 / 特产 CRUD、图片上传、上下架
- 社区管理 ：游记审核、评论管理、违规内容处理
- 订单管理 ：订单分类查看、状态更新、数据查询
- 用户管理 ：用户列表、禁用/启用、信息查看
- 数据看板 ：基础统计、趋势图表（ECharts）


项目亮点
自动获取景点的天气情况
引入地图api，获取景点的地图情况
支持对景点、酒店、特产、美食进行预订点赞评论
支持ai辅助自动进行行程规划
聊天室功能，大家可以探讨旅游计划
混合推荐策略
